# MongoDB NoSQL Database Management & Queries

## 📌 About the Project
This repository contains scripts and queries demonstrating practical skills in managing NoSQL databases using MongoDB. As modern automotive and web applications generate massive amounts of unstructured data (e.g., telemetry, sensor logs), understanding non-relational databases is a crucial asset for scalable software engineering.

## ⚙️ Core Features
* **CRUD Operations:** Implementation of Create, Read, Update, and Delete operations specific to document-oriented databases.
* **Data Aggregation:** Complex querying and data filtering to extract meaningful insights from JSON-like document structures.
* **Database Optimization:** Concepts of indexing and structuring NoSQL databases for high-performance retrieval.

## 🛠️ Tech Stack & Concepts
* **Technology:** MongoDB, NoSQL
* **Concepts:** Document Databases, Big Data Management, JSON/BSON manipulation, Database Querying.

## 📁 Repository Structure
* `lab12mongodb.txt` - Contains the MongoDB queries, aggregation pipelines, and database commands developed during the laboratory session.
